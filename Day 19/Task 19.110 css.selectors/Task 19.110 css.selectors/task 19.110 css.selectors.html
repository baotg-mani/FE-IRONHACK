Dùng chỉ dẫn trên với selectors, và dùng CSS style để
* Để cho tất cả element có chữ đậm, gợi ý font-weight
* tất cả các button, có chiều cao 20px. Gợi ý: selector button element
* tất cả paragraph, có chữ ngiêng. Gợi ý: selector paragraph element
* paragraph 1, 3, 5 có nền orange, gợi ý: chọn class trong paragraph
* paragraph 2, 4, 6 có nền green, gợi ý: class trong paragraph
* paragraph 4 có border màu đỏ, dotted, 2px; gợi ý border, chọn riêng item
* Button green có nền green, button red có nền red;
